（21）：tree

 

命令简介

tree 命令的作用是以树形结构显示目录下的内容。

常用的Linux发行版系统中默认没有这个命令，你需要通过安装才可以使用：

  1. #Centos
  2. yum install tree -y
  3. #Ubuntu
  4. sudo apt-get install tree

安装完成之后就可以正常使用这个tree命令了。

语法格式

tree [选项] [目录]

选项说明

  1. -a   #显示所有文件
  2. -d   #只显示目录（名称）
  3. -l   #显示链接文件的原始文件
  4. -f   #显示所列出的文件或目录的完整目录路径
  5. -i   #不以阶梯的形式显示文件或目录名称
  6. -q   #将控制字符以?字符代替，显示文件和目录名称
  7. -N   #直接显示文件或目录的名称
  8. -p   #显示每个文件的权限信息
  9. -u   #显示文件所有者或者uid
  10. -g   #显示文件所属组或者gid
  11. -s   #显示每个文件的大小信息
  12. -h   #以可读的方式显示文件的大小信息
  13. -D   #显示最后修改日期
  14. -v   #按字母数字正序显示文件
  15. -r   #按字母数字倒序显示文件
  16. -t   #按最后时间排序显示文件
  17. -C   #在文件和目录列表上加上色彩，便于区分文件类型
  18. -P pattern    #只显示匹配正则表式的文件或目录名称
  19. -I pattern    #与上结果相反

（22）：pwd

命令简介

pwd (print working directory)显示用户当前工作目录的绝对路径。

pwd 命令将当前用户的工作目录的全路径名称（从根目录）以绝对路径的方式标准输出在屏幕上。

语法格式

  1. tree [选项] [.....]
  2. pwd  [OPTION]...

选项说明

  1. -P  #打印当前目录的完整路径信息
  2. --help  #显示帮助信息
  3. --version #显示版本信息

应用举例

  1. #打印当前工作目录路径
  2. [root@centos7 dir]# pwd
  3. /root/testdir/dir
  4. #下面就是有链接文件
  5. [root@centos7 var]# ll mail
  6. lrwxrwxrwx. 1 root root 10 Aug 20 10:31 mail -> spool/mail
  7. [root@centos7 mail]# pwd
  8. /var/mail
  9. [root@centos7 mail]# pwd -P
  10. /var/spool/mail

pwd命令非常简单，基本上只会用到打印当前工作目录的功能，所以掌握起来很容易，同cd/ls/shutdown等一些命令一样，属于入门的最基础命令。

（24）：chattr/lsattr

命令简介

chattr 用来改变文件（扩展）属性。通常我们叫这个属性为特殊属性。

lsattr 查看文件扩展属性。

这种特殊属性有以下8种模式：

  1. a #限定文件具有某种功能。这个对于日志文件非常有作用，比如，不允许删除，只允许向里面追加内容。
  2. b #不更新或修改文件或目录的最后存储时间
  3. c #将文件或目录压缩后存储
  4. d #将文件或目录排除在倾倒操作之外
  5. i #锁定文件或目录，不被更改
  6. s #秘密删除文件或目录
  7. S #动态实时更新文件或目录
  8. u #防止文件或目录被意外删除

如果文件或目录被配置了上述8种之一的扩展属性，通过lsattr命令可以查看到：

  1. [root@centos7 testdir]# lsattr
  2. ----i----------- ./dir
  3. #结果说明这个目录被配置扩展属性
  4. [root@centos7 testdir]# chattr -i dir
  5. [root@centos7 testdir]# lsattr
  6. ---------------- ./dir

语法格式

  1. chattr [选项] mode file
  2. lsattr [选项] file

选项说明

chattr选项说明

  1. -R  #递归处理，将目录下的所有文件及子目录都加上此扩展属性
  2. -v<版本编号> #设置文件或目录版本号
  3. -V  #显示指令执行过程；
  4. + <属性> #添加文件或目录的某个扩展属性
  5. - <属性> #解除文件或目录的某个扩展属性
  6. = <属性> #指定某个扩展性至指定的文件或目录

lsattr选项说明

  1. -E  #显示属性的当前值
  2. -D  #显示扩展属性的名称
  3. -R  #递归
  4. -V  #显示版本信息
  5. -a  #列出目录下所有文件，包括隐藏文件
  6. #注意-E -D -R三个选项是不可以一起使用的，它们是相互排斥的。

应用举例

  1. #防止误删除的作用
  2. [root@centos7 testdir]# chattr +a test2.txt
  3. [root@centos7 testdir]# lsattr test2.txt
  4. -----a---------- test2.txt
  5. #不允许直接替换，只允许追加内容
  6. [root@centos7 testdir]# echo "12345">test2.txt
  7. -bash: test2.txt: Operation not permitted  
  8.  
  9. #查看文件或目录有没有扩展属性
  10. [root@centos7 testdir]# echo "12345">>test2.txt
  11. [root@centos7 testdir]# lsattr
  12. ---------------- ./dir
  13. ---------------- ./test2.txt~
  14. -----a---------- ./test2.txt
  15. lsattr: Operation not supported While reading flags on ./cp

这两个命令在日常工作也会常用到，锁定文件，或者当文件或系统出现故障时用于排查。

  1. [root@centos7 testdir]# lsattr
  2. ---------------- ./dir
  3. ---------------- ./test2.txt~
  4. ----i----------- ./test2.txt
  5. lsattr: Operation not supported While reading flags on ./cp
  6. [root@centos7 testdir]# echo "12345">>test2.txt
  7. -bash: test2.txt: Permission denied

比如上述情况，如果此文件是一个要实时存储数据的文件，被锁定了，那么就可能会引发应用运行故障，所以，这时就可以通过lsattr命令来查看与排查。

 

（25）：more

命令简介

more 分页显示文件内容。

以全屏的方式来显示文件的内容，每次显示一页，可按键向下翻页。more 命令中自带了很多快捷键，H（获得帮助信息），Enter（向下翻一行），空格（向下滚一屏），Q（退出）。

当你打开一个文件时，可以使用下面的快捷键进行相关的操作：

  1. Space（空格）键  #向下翻动一页
  2. Enter（回车）键  #显示文本的下一行
  3.  |    #输入一个模式，可以在当前文本内容查找一个相匹配的模式
  4. H 键  #显示帮助信息
  5. B 键  #向上翻一页
  6. Q 键  #退出

命令格式

  1. more [选项] [文件名]
  2. more [option] [file ...]

选项说明

  1. -<数字>  #指定每页显示的行数
  2. -c  #不进行滚屏操作
  3. -s  #将多个空行压缩成一行显示
  4. -u  #去除下划线
  5. +<数字>  #从指定的行开始显示

应用举例

  1. #查看文件mingongge的内容，在显示之前先清屏并且显示当前页数的百分比
  2. more -dc mingongge
  3.  
  4. #显示文件mingongge的内容，每100行显示一次，显示之前清屏
  5. more -c -100 mingongge
  6.  
  7. #从第十行开始显示mingongge.txt文件内容
  8. more +10 mingongge.txt
  9.  
  10. #从包含字符串 "mingonggee" 行开始显示mingongge.txt文件内容
  11. more +/"mingonggee" mingonggee.txt
  12.  
  13. #用ls打印当前目录列表，配合more分页显示
  14. ls | more

常用的用法基本就这些，也属于比较常用且简单的命令。

（26）：less

命令简介

less 命令分页上下翻页浏览文件内容。

less 命令与 more 命令相似，都可以用来查看文件内容，不同的是 less 命令可以向前或向后查看文件，而 more 命令只能向前查看。

用 less 命令显示文件时，按 PageUp 键向上翻页，按PageDown 键向下翻页，按 Q 键退出查看。

命令格式

  1. less [选项] [文件名]
  2. less [option] [file ...]

选项说明

  1. -e  #文件内容显示完成后，自动退出
  2. -f  #强制显示文件
  3. -g  #不加亮显示
  4. -l  #忽略大小写
  5. -N  #行首显示行号
  6. -s  #合并多个空行只显示一行
  7. -S  #不换行显示内容
  8. -x<数字>   #将TAB字符显示为指定个数的空格字符

应用举例

  1. #分页显示mingongge.log文件内容
  2. less /var/log/mingongge.log
  3.  
  4. #显示文件mingongge的内容，并在每行行首加上行号
  5. less -N mingongge.txt
  6.  
  7. #下/上一页：
  8. <Space> (down), b (up)
  9.  
  10. #less 命令转到文件的末尾/开始：
  11. G (end), g (start)
  12.  
  13. #向前查找字符串（按 n N 跳转到下一个/上一个）
  14. /mingongge
  15. #向后查找字符串（按 n N 跳转到下一个/上一个）
  16. ?mingongge
  17.  
  18. #退出 less 命令
  19. q

常用的用法基本就这些，与more命令一样，都属于比较常用且简单的命令。

（28）：ln

命令简介

ln 命令用于创建（软/硬）链接文件。

在linux系统中，有两种链接类型：硬链接与软件，默认创建的就是硬链接，创建软链接需要-s选项来配合完成。

硬链接文件是指通过索引节点来进行链接，在Linux系统中多个文件同时指向同一个索引节点，这种情况下的文件被称为硬链接文件。

软链接文件也称做符号链接（同Windows系统中快捷方式）。实际上它是一个文本文件，文本文件里存储着指向源文件链接的位置信息。

命令格式

  1. ln [选项] [链接文件名]
  2. ln [OPTION] [LINKNAME]

选项说明

  1. -b #创建备份文件
  2. -f #强行删除任何已存在的目标文件
  3. -i #覆盖现有文件前先询问用户
  4. -s #给一个文件创建软链接
  5. -v #打印每个链接文件的名称
  6. --help     #打印帮助信息后退出
  7. --version  #打印版本信息后退出

diff 命令用于查找、分析两个文件中不同的行，并打印输出在屏幕上。

  2. 此种模式下，可能通过移动光标，对字符或行进行删除操作。

（40）：gzip

命令简介

gzip 命令用来压缩文件，gzip，gunzip命令用于压缩或扩展 GNU GZIP 格式的文件。

如果压缩文件名对其文件系统而言太长，则gzip会将其截断。默认情况下，gzip 会将原始文件名和时间戳等信息保留在新产生的压缩文件中。

语法格式

  1. gzip [ OPTIONS ]  [ name ]
  2. gunzip [ OPTIONS ] [ name ]

选项说明

  1. -a  #使用ASCII文字模式
  2. -c  #将输出写入到标准输出，保持原始文件不变。
  3. -d  #解压缩文件
  4. -f  #强制压缩文件
  5. -h  #显示在线帮助信息
  6. -l  #列出压缩文件的相关信息
  7. -L  #显示版本与版权信息
  8. -n  #压缩文件时，不保留原来的文件名称及时间戳等属性信息
  9. -N  #压缩文件时，保留原来的文件名称及时间戳属性信息
  10. -q  #不显示警告信息
  11. -r  #递归处理，将指定目录下的所有文件及子目录一并处理
  12. -t  #测试压缩文件是否正确无误
  13. -v  #显示命令执行过程信息
  14. -V  #显示版本信息
  15. -<压缩效率>  #压缩效率是一个介于1~9的数值，默认值为“6”，指定的值越高，压缩率就越高
