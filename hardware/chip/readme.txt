1955年，“晶体管之父”的肖克利创办公司（肖克利实验室股份有限公司）
1957年：硅谷八叛将，成立了仙童半导体（Fairchild），两个发明：制造半导体的工艺，集成电路（晶体管、电阻、电容和电感）
1968年：诺依斯（N. Noyce）和摩尔（R.Moore）从仙童离职创办intel。
1969年：杰里·桑德斯（J. Sanders）从仙童离职创办AMD。
        其他：美国国家半导体、Altera

1947年，美国贝尔实验室发明了晶体管。
1958年，杰克·基尔比和罗伯特·诺伊斯发明了第一个集成电路（1个晶体管和几个电阻）。
1961年，美国德州仪器公司（TI）推出第一个商业化的集成电路。
1968年，戈登·摩尔和罗伯特·诺伊斯在硅谷创办了英特尔公司。
1971年，英特尔公司推出第一款微处理器芯片4位Intel4004（2250个晶体管），晶体管之间的距离为10微米，每秒运行6万次，频率为108KHZ。每个售价200美元。
1972年：8008芯片（3500个晶体管，8位），频率为200KHz，10微米。
1974年：8080芯片，集成6000个晶体管，运行速度2MHz，每秒执行29万条指令，6微米。
1978年，英特尔公司推出16位8086（29000个晶体管，3微米），时钟频率4.77MHZ，数据总线16位，地址总线20位，可寻址1MB内存。
1979年，英特尔公司推出16位8088（8086改良版，8位总线（节省资金））。
1982年，英特尔公司推出80286（13.4万个晶体管，1.5微米），时钟频率6MHZ，数据总线16位，地址总线24位，可寻址16MB内存，有实模式和保护模式，第一款带有MMU。
1985年，英特尔公司推出第一款32位微处理器芯片Intel80386（27.5万个晶体管，1.5微米），时钟频率16MHZ，数据总线32位，地址总线32位，可寻址4GB内容。
1989年，英特尔公司推出80486（120万个晶体管，25MHz，1微米，增加浮点处理器），上市价格950美元。
1993年，英特尔公司推出Pentium(奔腾)（80586， 310万个晶体管，0.8微米，时钟频率66MHZ）。
1997年，PentiumII处理器（300MHz，750万个晶体管，0.25微米）。 
1998年，Celeron(赛扬)处理器（低价）。 备注：赛扬300A是明星产品， Xeon（至强）用于服务器。
1999年，Intel Pentium III处理器（500MHz，950万，0.18微米，达到1GHz）     1999年，AMD推出64位CPU（AMD64、X86-64）
2000年，Intel Pentium 4处理器（1.5GHz，4200万，0.18微米），开始支持64位。

intel 4004最初是专门给日本一家名位Busicom的公司设计的，用于该公司的计算器产品。认识到cpu潜力后，intel用60000美元换回了微处理器设计的所有权。
Inte还曾开发了4001(40字节动态内存DRAM)、4002（只读存储器EPROM）、4003（10位移位Register）、4008（地址锁存器）、4009（I/O接口）
MCS-4： 4001+4002+4003+4004+键盘（输入）和数码管（输出）

推出4004芯片后，业务反应平淡，一年后intel推出8008（并获得处理器指令技术），获得业界关注，并于1974年被一款名位Mark-8（第一批加用计算机）的设备采用。
8008芯片原本位Datapoint公司设计，但这家公司最终没有足够财力支持费用，于是intel拥有了该芯片的知识产权，并获得了Datapoint公司开发的指令集。

SIMM：Single In-line Memory Module,单列内存模组
       wang Laboratories于1983年发明了单内联存储模块（SIMM）
	   单列直插式内存模块（single in-line memory module，缩写SIMM）
	   SIMM根据JEDEC JESD-21C标准进行了标准化。
	   容量：1MB-128MB

存储介子发展历史：
1950年前，穿孔卡片、穿孔纸带。
1950年，磁鼓存储器（约10KB），硬盘的前身
磁带存储器，磁带机/盒式磁带机（百MB）
1969年：软盘（8/5.25/3.5英寸   80KB/320KB/1.44MB）
1956年：硬盘存储器    IBM 350， 3.75MB，超过1吨
                      1961年， IBM 1301是第一个采用浮动磁头的硬盘，28MB， 体积也很大（39英寸），很贵。
                      1973年： 33FD，8英寸；Winchester硬盘（现代硬盘的原型），30MB
1971年：IBM发布了第一张只读的8英寸软盘（80KB）。1972年，Memore公司推出可读/写的软盘（174KB），1976年，5.25英寸软盘问世，1980年，索尼开发了3.5英寸软盘（1.44MB）
1976年：希捷的前身Shugart Associates公司和王安电脑联合开发了5.25英寸软盘驱动器（餐巾纸大小，但这个尺寸的硬盘是1980年才推出）
1980年：希捷推出5.25“的小硬盘，硬盘才开始规模进入微机领域
1982年：索尼和飞利浦发布了世界上第一部CD音频播放器CDP-101，光盘开始普及。
1980年：东芝发明了NAND Flash闪存；1988年，intel推出第一款商用NOR Flash芯片；1989年，东芝发布了世界上第一个NAND Flash产品。

2005年，8-Gb NAND 闪存芯片问世；2007年，3D闪存技术问世。SLC->MLC->TLC->QLC；
                                                        堆叠层数
2009年，英特尔和美光推出34nm TLC NAND。 固态硬盘大约从2010年开始普及。固态硬盘（SolidStateDisk），简称SSD，主要以NAND Flash为主。
2012年，三星创造了3D NAND，推出第一代3D NAND闪存芯片（32层SLC）。
2017年，SK海力士发售72层3D NAND。


目前幸存下来只有三家：希捷、东芝和西数；对机械硬盘来说，现在市场只有3.5”和2.5”的形态了
